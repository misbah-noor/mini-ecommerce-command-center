# 🛒 Mini Ecommerce Command Center

A modern full-stack ecommerce command center with Admin and User roles.  
Built using React, Node.js, Express, and MongoDB.

---

## 🚀 Features

- User Login System
- Role-Based Access (Admin / User)
- Admin Dashboard
- Add & Delete Products
- Cart System
- Orders Page
- Stripe payment simulation
- Search Functionality
- Light / Dark Theme Toggle
- Responsive & Modern UI

---

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Zustand
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## ⚙️ Local Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/misbah-noor/mini-ecommerce-command-center.git
```

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
